Purge
Orchid.Core.PostCommand
Purge document by Post Command.
Post commands can only take the current document to be run once. Re-run the graph to re-activate the post command. To get back the "Purge" window inside Revit, set the "activate" value to "false" and re-run the graph once more!
Input:
bool
activate 		> Set to true to activate (and false to deactivate)
