ByKeysValues
Orchid.Core.Dictionary

SetValueAtKeys
Orchid.Core.Dictionary
Produce a new Dictionary with a list of keys set to the new values, possibly overwriting existing key-value pairs. These two lists are expected to be of the same length. If not, the shorter of the two bounds the number of insertions.

NOTE: this node is being deprecated in the Orchid 2.0.x series and replaced by the OOTB nodes! The depreciation will be handled by migration!
Input:
Dictionary
dictionary 		> The Dictionary
Input:
List<string>
keys 		> The keys in the Dictionary to set. If the same key already exists, the value at that key will be modified.
Input:
List<object>
values 		> The corresponding values to insert.
Output:
Dictionary
dictionary 		> A new Dictionary with the entries inserted.
